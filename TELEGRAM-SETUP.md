# Налаштування Telegram-сповіщень
## ДІМ КЕРАТИНУ — Angelina Yurkevich hair studio

Після налаштування:
- **Ангеліна і Софія** отримують сповіщення про кожен новий запис
- **Клієнт** отримує підтвердження (якщо вказав @username в формі)

Займає ~10 хвилин.

---

## Крок 1 — Створити бота

1. Відкрий Telegram → знайди **@BotFather**
2. Напиши: `/newbot`
3. Введи назву, наприклад: `Дім Кератину`
4. Введи username (лише латиниця + `bot` в кінці): наприклад `dim_keratin_bot`
5. BotFather надішле **токен** — виглядає так:
   ```
   7412345678:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   Скопіюй і збережи.

---

## Крок 2 — Отримати Chat ID Ангеліни

1. Знайди бота в Telegram за його username (`@dim_keratin_bot`)
2. Натисни **START**
3. Відкрий у браузері (підстав свій токен):
   ```
   https://api.telegram.org/bot7412345678:AAHxxx.../getUpdates
   ```
4. Знайди `"chat": { "id": 123456789 }` — це і є **Chat ID**

---

## Крок 3 — Отримати Chat ID Софії

Те саме: Софія знаходить бота, натискає START → отримуєш її chat_id через getUpdates.

---

## Крок 4 — Вставити в config.js

Відкрий файл `js/config.js` і заміни:

```javascript
const BOT_TOKEN = 'ВАШ_ТОКЕН_БОТ';   // ← токен з кроку 1

const RECIPIENTS = [
  { name: 'Ангеліна', chat_id: 'CHAT_ID_ANGELINA' },  // ← chat_id Ангеліни
  { name: 'Софія',    chat_id: 'CHAT_ID_SOFIA'    }   // ← chat_id Софії
];

const BOT_USERNAME = 'dim_keratin_bot';  // ← username бота (без @)
```

Наприклад:
```javascript
const BOT_TOKEN = '7412345678:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx';

const RECIPIENTS = [
  { name: 'Ангеліна', chat_id: '111222333' },
  { name: 'Софія',    chat_id: '444555666' }
];

const BOT_USERNAME = 'dim_keratin_bot';
```

```

📞 Телефон: +38 073 000 00 00
💬 Telegram: @olya_test

📅 Дата: вівторок, 12 травня
🕐 Час: 11:00

```

```
💇 Послуга: Кератинове випрямлення

До зустрічі в ДІМ КЕРАТИНУ! ✨
```

## Управління розкладом

Розклад роботи знаходиться в `js/config.js`:

```javascript
workDays: [1, 2, 3, 4, 5, 6],  // Пн-Сб (змінювати цифри: 0=Нд...6=Сб)

timeSlots: ['09:00', '10:00', ...],  // стандартні години

holidays: ['2026-05-09'],  // вихідні/свята — клієнти не зможуть записатись

specialDays: {
  '2026-05-10': ['10:00', '11:00']  // особливий розклад на конкретний день
}
```

**Щоб додати вихідний день:** дописати дату в `holidays`
**Щоб змінити години в конкретний день:** дописати в `specialDays`
